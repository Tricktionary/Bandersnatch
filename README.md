# Bandersnatch

### What is this? 
A re-make of netflix's bandersnatch game as a web app created with Django and Vue

### Getting Started

- `source env/bin/activate`
- `python3 manage.py runserver`
- `python3 manage.py makemigrations`
- `python3 manage.py migrate`
